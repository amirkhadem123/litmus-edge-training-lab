# requirements.txt — Python dependencies for Litmus Lab
#
# HOW TO READ THIS FILE:
# Each line is a package name with a minimum version (>=) or exact version (==).
# "==" pins to an exact version so the build is reproducible.
# ">=" allows newer compatible versions.
#
# The litmussdk package is NOT listed here because it is installed from the
# local wheel file during the Docker build (see Dockerfile). Listing it here
# would cause pip to try to fetch it from PyPI, where it is not available.
